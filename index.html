Hospedagem GitHub

para criar um site no GitHyb.com devemos ter uma conta válida e um repositório com um arquivo index.html

Hospedando um site

  1. Criar um repositório;
  2. Publicar o repositório utilizando o comando publish;
  3. Abrir o repositório no VScode;
  4. Criar o arquivo index.html
  5. Realizar o commit, criando assim, uma nova versão;
  6. Enviar para o repositório remoto usando push;
  7. Entrar no site github.com, acessar o repositório, vai em Configurações/Settings, escolho Páginas/Pages, após isso, escolhe-se a branch em Source;
  8. E por último, clicar na URL gerada.

    9. Branches

    Branches são ramificações do seu projeto, os projetos em geral ficam na branch main, a branch main deve ter a função final e funcional do seu projeto, o projeto pode ter branches secundárias, Clique aqui para saber mais sobre branches secundárias
